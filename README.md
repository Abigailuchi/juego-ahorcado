# juego-ahorcado
Proyecto Autónomo 2 

Juego del Ahorcado
Descripción

Este proyecto implementa el clásico juego del Ahorcado en Python. El jugador debe adivinar una palabra secreta letra por letra antes de quedarse sin intentos.

Objetivo

Aplicar los conocimientos de Lógica de Programación mediante el uso de:

Estructuras condicionales (if, else).
Bucles (while).
Funciones.
Conjuntos (set).
Control de versiones con Git y GitHub.
Archivos del proyecto
ahorcado.py → Código fuente del juego.
diagrama_flujo.png → Diagrama de flujo utilizado para diseñar la lógica.
README.md → Documentación del proyecto.
Requisitos
Python 3.x
Cómo ejecutar el programa

Descargar o clonar el repositorio:

git clone https://github.com/Abigailuchi/juego-ahorcado
Entrar a la carpeta del proyecto.

Ejecutar:

python ahorcado.py
Funcionamiento
El sistema selecciona una palabra al azar.
La palabra se muestra oculta con guiones.
El jugador ingresa letras.
Si la letra es correcta, se revela.
Si es incorrecta, se pierde un intento.
El juego termina cuando el jugador adivina toda la palabra o se queda sin intentos.
